# Author workflows

How content authors create, edit, and prepare documentation for publication.

---

## Create new content

1. **Upload** — add source file (FM, DITA, MD, DOCX)
2. **Edit** — open in DocEditor for in-platform changes
3. **Save version** — DocManager tracks version history

---

## Edit existing content

1. Open document in **DocEditor**
2. Make changes with autosave
3. Add change summary for reviewers
4. Submit for review when ready

---

## Best practices

- Use consistent file naming conventions
- Add metadata tags at upload time
- Run validation before submitting for review
- Link related topics in DITA maps
