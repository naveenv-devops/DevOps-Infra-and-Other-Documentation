# Review & approval

Quality gates before content is published to production channels.

---

## Review workflow

```
Author submits → Reviewer assigned → Feedback / Approve → Publish
```

---

## For authors

1. Complete conversion and fix validation errors
2. Click **Submit for review**
3. Add summary of changes
4. Address reviewer feedback and resubmit if needed

---

## For reviewers

1. Open review queue in Dashboard
2. Compare source and output
3. Check validation report
4. **Approve** or **Request changes** with comments

---

## Approval requirements

Workspace admins can configure:

- Minimum number of reviewers
- Required reviewer roles
- Auto-publish on approval vs manual publish
