# Glossary

Met-R platform terminology.

| Term | Definition |
|---|---|
| **Converter** | Microservice that transforms content between formats |
| **DITA** | Darwin Information Typing Architecture — XML standard for technical docs |
| **DocSite** | See Infinity DocSite |
| **Determinacy Engine** | Rule-based engine for content processing decisions |
| **DITA-OT** | DITA Open Toolkit — industry-standard publishing engine |
| **Infinity DocSite** | Met-R dynamic documentation portal |
| **Pipeline** | Sequence of conversion, validation, and publish steps |
| **PropDITA** | Proprietary DITA dialect requiring normalization |
| **Workspace** | Isolated tenant environment on Met-R |
| **XSLT** | Extensible Stylesheet Language Transformations |
