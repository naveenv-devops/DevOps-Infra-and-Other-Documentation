# Publishing & output

Deliver documentation to PDF, web, CMS, and AI-ready structured formats.

---

## Output services

| Service | Repository | Output |
|---|---|---|
| **PDF** | `metR-Output-PDF` | Print-ready PDF |
| **HTML5 / CMS** | `metR-Output-HTML5-CMS` | Web HTML, CMS packages |
| **XML / JSON AI** | `metR-Output-XML-JSON-AI` | Structured content for AI pipelines |
| **DocPublisher** | `metR-DocPublisher` | Publication orchestration |

---

## Publishing workflow

1. Content passes validation and review gates
2. DocPublisher selects output profile
3. Output service generates deliverable
4. Content delivered to channel (download, CMS push, DocSite)

---

## Output channels

| Channel | Best for |
|---|---|
| PDF | Print, regulated industries, offline distribution |
| HTML5 | Web help, responsive documentation |
| CMS | Sitecore, custom CMS integration |
| XML/JSON AI | RAG, search, AI agent consumption |
| Infinity DocSite | Searchable product documentation portal |
