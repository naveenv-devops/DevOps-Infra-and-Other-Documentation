# FrameMaker converter

Convert Adobe FrameMaker content to DITA and other structured formats.

**Repository:** `metR-Converter-FrameMaker`  
**Status:** GA

---

## Overview

The FrameMaker converter migrates legacy `.fm` and book files into structured DITA content, preserving topic structure, cross-references, and conditional processing attributes where possible.

---

## Supported inputs

| Input | Extensions |
|---|---|
| FrameMaker document | `.fm` |
| FrameMaker book | `.book` |

---

## Supported outputs

| Output | Format |
|---|---|
| DITA topics | `.dita` |
| DITA map | `.ditamap` |
| HTML (intermediate) | `.html` |

---

## AI-assisted conversion

FrameMaker-to-DITA workflows support AI agent-assisted conversion for complex legacy structures. See [AI capabilities](../ai/README.md).
