# HTML converter

Normalize and transform structured HTML content for downstream publishing.

**Repository:** `metR-Converter-HTML`  
**Status:** GA

---

## Overview

The HTML converter ingests structured HTML and applies Met-R transformation rules for migration into DITA, DocBook, or CMS-ready output.
