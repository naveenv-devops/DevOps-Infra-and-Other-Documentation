# Infinity DocSite

**Infinity DocSite** (`metR-InfinityDocSite`) delivers a dynamic, searchable documentation portal for your product content.

---

## Overview

Infinity DocSite publishes validated Met-R content as a branded documentation website with search, navigation, and version support.

---

## Features

| Feature | Description |
|---|---|
| **Full-text search** | Search across all published topics |
| **Versioned content** | Publish and browse multiple product versions |
| **Responsive design** | Mobile and desktop optimized |
| **Custom branding** | Workspace logo, colors, and domain |
| **Access control** | Public or authenticated access per workspace |

---

## Publishing to Infinity DocSite

1. Complete conversion and review workflow
2. In DocPublisher, select **Infinity DocSite** as output channel
3. Configure site map and navigation
4. Publish — site updates within minutes
