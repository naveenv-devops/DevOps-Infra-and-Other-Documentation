# Governance & security

Access control, audit, and compliance for Met-R workspaces.

---

## Access control

- Role-based access per workspace
- API keys scoped to workspace
- SSO integration (enterprise plans)

---

## Content security

- All traffic encrypted in transit (TLS 1.2+)
- Workspace data isolation between tenants
- Security scanning on all converter pipelines (`security-scan` check)

---

## Audit

- User action logs in Client Admin
- Conversion and publish job history
- API access logs via API Gateway

---

## Compliance

Contact MetaPercept for:

- Data residency requirements
- SOC 2 / ISO documentation
- Custom BAA or DPA agreements
