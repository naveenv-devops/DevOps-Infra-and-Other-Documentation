# FAQ

Frequently asked questions about Met-R.

## General

**What is Met-R?**  
Met-R is MetaPercept's documentation orchestration platform for converting, managing, and publishing technical content. See [Introduction](../get-started/introduction.md).

**Who is Met-R for?**  
Technical writers, documentation managers, DevOps teams, and developers working with structured content at scale.

**How do I get workspace access?**
