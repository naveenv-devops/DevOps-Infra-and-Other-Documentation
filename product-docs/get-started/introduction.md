# Introduction to Met-R

**Met-R** ([met-r.io](https://met-r.io)) is MetaPercept's intelligent documentation orchestration platform. It helps enterprises convert, manage, validate, and publish technical content across formats — from legacy authoring tools to modern web, PDF, and CMS delivery.

---

## Who is Met-R for?

| Audience | Use Met-R to… |
|---|---|
| **Technical writers** | Author, convert, and publish DITA, DocBook, FrameMaker, and Markdown content |
| **Documentation managers** | Govern workflows, approvals, and multi-channel output |
| **DevOps / platform teams** | Operate conversion pipelines, CI/CD, and infrastructure |
| **Developers** | Integrate via API Gateway, webhooks, and custom XSLT transforms |

---

## What problems does Met-R solve?

- **Format fragmentation** — unify content from FrameMaker, DITA, DocBook, Word, PDF, HTML, and CMS sources
- **Manual conversion** — automate repeatable transformation pipelines with validation
- **Publishing bottlenecks** — deliver PDF, HTML5, CMS, and structured XML/JSON from a single source
- **Quality risk** — enforce schema validation, XSLT checks, and review gates before publish
- **AI-ready content** — prepare and enrich documentation for AI agents and intelligent search
