# Workspace setup

Configure your Met-R workspace, users, and permissions.

---

## Create a workspace

Workspaces are provisioned by MetaPercept. Contact your account manager for new workspace creation.

---

## User roles

| Role | Permissions |
|---|---|
| **Viewer** | Read published content |
| **Author** | Upload, edit, convert |
| **Reviewer** | Approve content for publish |
| **Client Admin** | Manage users, pipelines, settings |
| **Server Admin** | Infrastructure and service config |

---

## Add users

1. Open **Client Admin** → **Users**
2. Click **Invite user**
3. Enter email and assign role
4. User receives invitation email
